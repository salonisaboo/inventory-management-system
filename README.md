# Smart Inventory Management System

Inventra is a full-stack Inventory Management System designed to help businesses manage products, suppliers, purchases, sales, and stock movement in a structured and secure manner.  
The system simplifies inventory operations, maintains accurate transactional records, and supports informed decision-making through organized data access.

---

## Project Overview

Inventra provides a centralized platform for handling end-to-end inventory workflows.  
It enables users to track stock availability, manage suppliers, record purchases and sales, and control system access using role-based authentication.

The application follows a layered backend architecture and communicates with the frontend using RESTful APIs to ensure scalability, maintainability, and clean separation of concerns.

---

## Features

### Inventory Management
- Product and category management  
- Supplier management and tracking  
- Purchase and sales transaction handling  
- Real-time stock updates and availability monitoring  

### Security and Access Control
- Secure authentication using JWT  
- Role-based access control for protected operations  

### User Interface
- Dashboard overview for inventory and transactions  
- Pagination, sorting, and filtering for large datasets  
- Responsive UI with reusable components
